# Vipre Prototypes

A collection of interactive Vipre product prototypes and explorations.

🔗 **Live site:** https://sixbase.github.io/vipre-prototypes/

## Prototypes

| Folder | What it is |
|--------|------------|
| `scope-navigator` | Scope navigator shell + MSP/Symphony shell variants |
| `symphony-shell` | Standalone Symphony shell prototype |
| `action-rules` | Action-rules prototype |
| `banner-modal` | Banner / modal prototype |
| `marketing-overview` | Marketing overview prototype |
| `vipre-prototypes` | Shared prototype assets |
| `vipre-audit` | Consistency audit — [live](https://sixbase.github.io/vipre-audit/) |

## Develop

Each prototype is its own app. `cd` into a folder, then:

```bash
npm install
npm run dev
```

Deployed to GitHub Pages on every push to `main`.
